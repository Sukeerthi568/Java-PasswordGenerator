# Advanced Password Generator (Java)

## Project Description

The **Advanced Password Generator** is a Java-based console application that generates secure and customizable passwords.
Users can choose the password length and select which character types to include such as uppercase letters, lowercase letters, numbers, and symbols.

The program also evaluates the **strength of the generated password** to help users create secure credentials.

---

## Features

* Generate random passwords
* Custom password length
* Option to include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Password strength checker
* Secure random generation using `SecureRandom`
* Simple and user-friendly console interface
